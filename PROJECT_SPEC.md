# hamanoie.com — プロジェクト仕様書

## 概要
「山と月と浜の家」のサイト。空き家リノベのアトリエスペースの名刺サイト。
「どんな場所か」「どこにあるか」が一目でわかることが目的。

## コンセプト
**「山と海のあいだの、小さな実験室。」**

手仕事、AI、対話、表現——そのとき関心のあることを持ち寄って、
小さく実験するための場所。何をやるかは決めすぎない。自由な器。
